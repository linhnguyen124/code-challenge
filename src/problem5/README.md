# Description

This is repo running with NodeJS

## Setup

**Requirement**:

- node >= 21.5.0
- docker >= 24.0.7
- docker-compose: >= v2.23.3-desktop.2
- npm => 10.7.0

After you clone the repo, please do following steps

```bash
make build_application

# development
$ npm install
$ npm run watch
$ npm run start
```
